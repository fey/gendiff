#

## prerequisites

* PHP 7.4
* Composer

## installation

`composer g require fey/gendiff dev-master`

## usage

```sh
Generate diff

Usage:
  gendiff (-h|--help)
  gendiff [--format <fmt>] <firstFile> <secondFile>

Options:
  -h --help                     Show this screen
  --format <fmt>                Report format [default: pretty]
```
